# work-manager-android
<p>This application just for practice how to use Work Manager in android studio use Java code,
  and the application using API for fetch data.</p>
<h3>Features</h3>
<ul>
  <li>Room Database</li>
  <li>Retrofit</li>
  <li>MVVM architecture component</li>
  <li>Recycler view</li>
</ul>
